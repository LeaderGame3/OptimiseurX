⚡ Optimiseur X

Optimiseur X est un outil d'optimisation tout-en-un pour Windows 10 et 11, conçu pour améliorer les performances de votre système en quelques clics.


🖥️ Aperçu

Optimiseur X offre une interface graphique moderne (WPF) avec plus de 12 catégories d'optimisations :

| ⚙️ Avancé (Expert) | Options avancées pour utilisateurs expérimentés |
| 💾 Disque | Optimisation des performances disque |
| 🎨 Interface visuelle | Allègement de l'interface Windows |
| 🎮 Jeux vidéo | Optimisations spécifiques au gaming |
| 📦 Logiciels | Gestion et suppression de bloatware |
| 🔧 Maintenance | Nettoyage et entretien du système |
| 🌐 Réseau Internet | Optimisation de la connexion réseau |
| ♻️ Restauration | Annulation des modifications appliquées |
| 🛠️ Services système | Désactivation des services inutiles |
| 💿 Stockage et NVMe | Optimisations SSD/NVMe |
| 🖥️ Système d'exploitation | Réglages du système |
| 🔒 Vie privée | Protection de la confidentialité |


🚀 Installation et utilisation

1. Téléchargez le fichier "OptimiseurX.exe"
2. Clic droit → "Exécuter en tant qu'administrateur"
3. Sélectionnez les optimisations souhaitées
4. Cliquez sur le bouton "APPLIQUER LA SÉLECTION"

> IMPORTANT!
> Les droits d'administrateur sont requis pour appliquer les modifications au registre et aux services système.


🛡️ Fonctionnalités clés

- Interface graphique moderne — Design sombre avec thème cyan, surveillance en temps réel du CPU, RAM, disque et GPU
- Mode Expert — Affiche le code exécuté dans les infobulles pour transparence totale
- Barre de recherche — Trouvez rapidement une optimisation parmi toutes les catégories
- Rapport d'optimisation — Génère un fichier `Rapport_Optimisation.txt` détaillant chaque action
- Annulation granulaire — Fichier `Annulation_Registre.csv` permettant de restaurer les paramètres modifiés
- Détection automatique — Reconnaissance SSD/NVMe, RAID, domaine réseau et version Windows


⚠️ Avertissement

> ATTENTION!
> Cet outil modifie des paramètres du système, des clés de registre et des services Windows. Utilisez-le à vos propres risques. Il est recommandé de créer un point de restauration système avant toute utilisation.
